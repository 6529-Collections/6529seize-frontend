# PR Report: Home Mint Subscription Awareness

PR: https://github.com/6529-Collections/6529seize-frontend/pull/2984
Branch: codex/home-subscription-awareness -> main
Related PRs: None

## Summary

Home and current The Memes detail pages now surface The Memes subscription affordance more broadly so connected users can see subscription controls or profile links even when they are not already subscribed, while disconnected or proxy-context users see a non-actionable awareness state.

## What Changed

- Latest Drop passes the displayed Meme Card id into an awareness-only subscription widget, avoiding the upcoming-meme status endpoint for current or already-dropped cards.
- Next Drop keeps the canonical next mint behavior and continues to show the subscription widget in the top panel.
- The current/latest `/the-memes/{id}` detail page shows the same awareness-only subscription row below the mint countdown.
- Awareness-only rows use a read-only disabled switch for active/current drops or disconnected visitors, omit the passive `Manage in profile` text, and move profile access to an icon-only link beside the question-mark info link.
- The subscription widget renders a profileless/proxy awareness row with a subscription info link and, when available, a profile subscriptions icon link.
- Compact subscription rows can show optional subscription info and profile subscription links.
- `/about/subscriptions` shows connected users a primary `My subscriptions` action beside the about-section dropdown, and shows disconnected visitors a primary connect-wallet action that forwards them to profile subscriptions after connection.
- Home docs now mention the expanded subscription affordance.

## Frontend Impact

- Routes/views: `/` home Latest Drop and Next Drop mint panels, current `/the-memes/{id}` detail pages, and `/about/subscriptions`.
- Components: home mint subscription widget, compact Meme subscription row, Meme detail page mint panel, and about contents navigation.
- Generated API/client: Not affected.
- User-visible behavior: subscription awareness is shown to more visitors, and connected users can reach `/{user}/subscriptions` from the home mint panel.

## Config / Env

No config or environment changes.

## Release Notes

The home mint panel and current The Memes detail page now make The Memes subscriptions more discoverable for connected, unsubscribed, disconnected, and proxy-context visitors, while preserving the existing platform/country hide rule and subscription action disablement rules.
