# PR Report: Home Mint Subscription Awareness

PR: https://github.com/6529-Collections/6529seize-frontend/pull/2984
Branch: codex/home-subscription-awareness -> main
Related PRs: None

## Summary

Home now surfaces The Memes subscription affordance more broadly in the mint panel so connected users can see subscription controls even when they are not already subscribed, while disconnected or proxy-context users see a non-actionable awareness state.

## What Changed

- Latest Drop passes the displayed Meme Card id into the subscription widget instead of using the canonical next mint id.
- Next Drop keeps the canonical next mint behavior and continues to show the subscription widget in the top panel.
- The subscription widget renders a profileless/proxy awareness row with a subscription info link and, when available, a profile subscriptions link.
- Compact subscription rows can show optional subscription info and profile subscription links.
- Home docs and Help Bot records now mention the subscription affordance.

## Frontend Impact

- Routes/views: `/` home Latest Drop and Next Drop mint panels.
- Components: home mint subscription widget and compact Meme subscription row.
- Generated API/client: Not affected.
- User-visible behavior: subscription awareness is shown to more visitors, and connected users can reach `/{user}/subscriptions` from the home mint panel.

## Config / Env

No config or environment changes.

## Release Notes

The home mint panel now makes The Memes subscriptions more discoverable for connected, unsubscribed, disconnected, and proxy-context visitors, while preserving the existing platform/country hide rule and subscription action disablement rules.
