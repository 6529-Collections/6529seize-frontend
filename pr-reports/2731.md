## Task Summary

Restore `NewVersionToast` to the version immediately before the broad toast
redesign changed its copy and styling, while leaving all other toast changes
intact.

## Change Summary

- Restored `components/utils/NewVersionToast.tsx` to the standalone
  Tailwind-styled prompt used before `Redesign app toast notifications`.
- Restored the visible `Yes, again!` copy, wink emoji, rocket artwork, app/non-app
  positioning, and hover glow treatment.
- Removed this component's dependency on shared `AppToast.module.scss`.
- Moved the restored visible copy and refresh accessible name into
  `i18n/messages/en-US.ts` so the touched UI uses the existing message fallback
  path while rendering the same English text.
- Updated the client message lookup to normalize the browser locale instead of
  hardcoding `en-US`.
- Updated `__tests__/components/utils/NewVersionToast.test.tsx` to assert the
  restored copy and app/non-app positioning.
