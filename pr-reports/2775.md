## Summary
Fixes the Network Nerd page on phone-width screens by preventing the filter controls, focus tabs, dropdowns, and dense leaderboard table from colliding or squeezing into unreadable columns.

## What Changed
- Stacked the Network Nerd toolbar controls on narrow screens.
- Converted the focus tabs into full-width, high-contrast mobile tap targets with scoped mobile styles that survive the base focus-button cascade.
- Let toolbar dropdown menus render outside the toolbar card instead of being clipped.
- Added mobile-only horizontal-scroll sizing for both leaderboard tables without changing the desktop table layout.
- Replaced the static text `>>` cue with scroll-aware double-chevron icon controls pinned above the table header: the right cue appears only when more content is available to the right, the left cue appears only after horizontal scrolling, and each cue hides at its scroll edge.

## Frontend Impact
- Routes/views: `/network/nerd`, `/network/nerd/cards-collected`, `/network/nerd/interactions`.
- Components: `components/leaderboard/Leaderboard.module.scss`, `components/leaderboard/LeaderboardTableScrollShell.tsx`, `components/leaderboard/LeaderboardCardsCollected.tsx`, `components/leaderboard/LeaderboardInteractions.tsx`.
- Generated API/client: Not affected.
- User-visible behavior: mobile users should see separated controls, unclipped dropdown menus, and hinted horizontally scrollable leaderboards instead of overlapped toolbar/table text; desktop keeps the normal fitted table layout.
