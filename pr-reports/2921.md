# PR Report: About contents dropdown navigation

PR: https://github.com/6529-Collections/6529seize-frontend/pull/2921
Branch: codex/about-contents-nav -> main
Related PRs: None

## Summary

Replaces the persistent About in-page side menu with a compact sticky current-page dropdown and adds a root `/about` index so users can browse About pages and related Network resources by category without a second side navigation.

## What Changed

- Added a shared About current-page dropdown that lists the requested About pages by category, highlights the current page, and navigates without changing existing `/about/{section}` paths.
- Added `/about` as a grouped index page with category cards for Collections, Delegation, Network, Resources, Community, and Legal pages.
- Added TDH, xTDH, Health, Definitions, Levels, and Network TDH to the About Network group while keeping their canonical `/network/*` paths.
- Reused the About contents dropdown on the requested Network information pages and gave the dropdown viewport padding so it no longer hugs the screen edge.
- Made the dropdown tolerate isolated route renders without a cookie-consent provider while still hiding Subscriptions conservatively for iOS when country is unknown.
- Added a “What is xTDH” link from `/xtdh` to the canonical xTDH explainer.
- Preserved the existing Subscriptions iOS/country visibility rule and kept Cookie Policy reachable from the About navigation.
- Added link-backed dropdown items without underlined link styling and kept the menu constrained within narrow viewports.
- Removed the desktop side-menu column and mobile duplicate About menu from the main About route and About Tech subroutes.
- Updated Primary Address documentation that referenced the previous in-page About menu.

## Frontend Impact

- Routes/views: `/about`, `/about/{section}`, `/about/tech/{reportSlug}`, `/about/tech/wallet-authentication`, `/xtdh`, `/network/tdh`, `/network/xtdh`, `/network/health`, `/network/definitions`, `/network/levels`, and `/network/health/network-tdh`.
- Components: About section layout, grouped About route data, root About index cards, Network information pages, and the sticky current-page dropdown built on the existing compact menu primitive.
- Generated API/client: Not affected.
- User-visible behavior: About pages now have the global sidebar plus main content, with a small top-right page selector instead of a persistent in-page side menu; `/about` now opens a category-based About landing page.

## Config / Env

No config or environment changes.

## Release Notes

About and related Network information pages now use a compact sticky current-page dropdown with categorized navigation, `/about` opens a grouped landing page of About links instead of a 404, and About includes a dedicated Network group whose links keep their canonical paths.
