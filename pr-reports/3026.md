# PR Report: About and Tools index polish

PR: https://github.com/6529-Collections/6529seize-frontend/pull/3026
Branch: codex/about-tools-link-polish -> main
Related PRs: #3024 (About menu IA cleanup, merged first; this PR was reworked on top of it)

## Summary
This PR tightens the `/about` and `/tools` landing indexes so both pages use the same spacing rhythm and card affordances: cards gain right-arrow affordances, `/tools` cards drop their left-side icons, and the `/about` Subscriptions card is renamed to `Subscription Minting`. The About index keeps the group structure and per-card descriptions introduced by #3024.

## What Changed
- Renamed the `/about` Collections & Minting card from `Subscriptions` to `Subscription Minting` (i18n label; group structure and ordering from #3024 kept as-is).
- Removed the left-side icons from `/tools` landing cards while preserving right arrows.
- Added matching right arrows to `/about` landing cards, alongside the per-card descriptions from #3024.
- Matched the `/tools` landing content container to the `/about` spacing.
- Shortened the `/tools` metadata description and aligned the Tools docs/help index records with the visible card behavior.
- Removed the now-unused icon and lead-paragraph plumbing from the shared grouped link index component, and the dead tools nav-item helper exports.

## Frontend Impact
- Routes/views: `/about` and `/tools` landing pages.
- Components: shared grouped link index, About index, Tools index, and their route metadata/copy sources.
- Generated API/client: Not affected.
- User-visible behavior: About and Tools cards are more visually consistent; `/tools` cards no longer show left icons; `/about` and `/tools` cards show right arrows; the About subscriptions entry reads `Subscription Minting`.

## Config / Env
None.

## Release Notes
- `/about` now labels the subscriptions entry as `Subscription Minting` and gives cards the same right-arrow affordance used by Tools.
- `/tools` now uses About-style landing spacing, removes card icons, and has the shorter metadata description.
