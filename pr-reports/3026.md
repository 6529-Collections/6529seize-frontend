# PR Report: About and Tools index polish

PR: https://github.com/6529-Collections/6529seize-frontend/pull/3026
Branch: codex/about-tools-link-polish -> main
Related PRs: None

## Summary
This PR tightens the `/about` and `/tools` landing indexes so both pages use the same spacing rhythm, remove the explanatory lead paragraph, and present simple label cards with right-arrow affordances.

## What Changed
- Renamed the `/about` Collections card from `Subscriptions` to `Subscription Minting` and moved it to the end of the group.
- Removed the left-side icons from `/tools` landing cards while preserving right arrows.
- Added matching right arrows to `/about` landing cards.
- Matched the `/tools` landing content container to the `/about` spacing.
- Shortened the `/tools` metadata description and aligned the Tools docs/help index records with the visible card behavior.

## Frontend Impact
- Routes/views: `/about` and `/tools` landing pages.
- Components: shared grouped link index, About index, Tools index, and their route metadata/copy sources.
- Generated API/client: Not affected.
- User-visible behavior: About and Tools cards are more visually consistent; the `/tools` lead copy is removed; `/tools` cards no longer show left icons; `/about` cards now show right arrows.

## Config / Env
None.

## Release Notes
- `/about` now labels the subscriptions entry as `Subscription Minting`, places it last in Collections, and gives cards the same right-arrow affordance used by Tools.
- `/tools` now uses About-style landing spacing, removes card icons and page lead copy, and has the shorter metadata description.
