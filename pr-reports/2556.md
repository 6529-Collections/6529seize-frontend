## Frontend Change Summary

- Updated frontend auth/session handling for auth v2 web/native compatibility.
- Added structured wallet signature generation paths and frontend rollout gating.
- Added connection sharing acceptance flow and associated auth/session utilities.
- Updated generated API models/client surfaces from backend auth v2 contract changes.
- Addressed generated-model static-analysis feedback, including unused generated imports.
- Replayed the local unpushed v1-auth migration logic onto the clean PR branch:
  - valid v1 JWT sessions now prompt users to upgrade to session v2 instead of immediately logging out;
  - users can defer the upgrade prompt for 2-hour reminder intervals while temporary access remains;
  - each wallet gets up to 72 hours from first prompt, capped by `SESSION_V2_MIGRATION_DEADLINE`;
  - non-signable active/shared auth connections show a reshare/update-required prompt instead of a sign button;
  - expired migration windows clear active auth and invalidate cached state.
- Added session-v2 marker persistence so refreshed/logged-in v2 sessions are distinguished from legacy v1 auth.
- Avoided cookie-only web session refresh/logout attempts when the frontend API endpoint is cross-origin.
- Switched websocket auth failure/timeout close codes to app-private close codes.
- Removed browser-rejected Permissions-Policy directives from security headers.

## Frontend Impact Summary

- Routes/views/components changed: connection sharing acceptance route, auth providers/context, v1-auth migration sign modal, wallet setup/adapter behavior, header/sidebar auth display paths, drop/signature-related flows.
- Generated models/API client changes: auth nonce/session/login/logout/refresh/connection-share generated models and API client surfaces.
- User-visible behavior changes: users can participate in the auth v2 wallet session and connection sharing flows once backend/env gates are enabled; legacy v1 sessions are prompted to migrate, can defer reminders while allowed, and are removed after their effective migration deadline.
