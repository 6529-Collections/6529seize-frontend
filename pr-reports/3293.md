# PR Report: Format Shared Pagination Page Counts

PR: https://github.com/6529-Collections/6529seize-frontend/pull/3293
Branch: `agent/format-pagination-page-counts` -> `main`
Related PRs: None

## Summary

Shared table pagination now displays localized copy and locale-formatted current and total page counts, including on the Network page.

## What Changed

- Routed both values in the shared pagination label through the existing integer formatter using the active browser locale.
- Added localized pagination labels for all supported locales.
- Added regression coverage for translated copy, word order, and locale-specific thousands separators.

## Frontend Impact

- Routes/views: Network, community members, profile activity, collected items, and user activity views that use the shared paginator.
- Components: `CommonTablePagination`.
- Generated API/client: Not affected.
- User-visible behavior: Pagination labels and number grouping follow the active supported browser locale, such as `Page 12,345 of 20,598` in English and `Seite 12.345 von 20.598` in German.

## Release Notes

Shared table pagination now uses localized labels and locale-appropriate thousands separators for large current-page and total-page values.
