# 6529SEIZE-FRONTEND

PORT: 3000

### Install

```
npm i
```

### Build

```
npm run build
```

### Environment

To run the project you need a .env file.

[Sample .env file](https://github.com/6529-Collections/6529seize-frontend/tree/main/.env.sample)

### Run

- Locally

```
npm run dev
```

- Production

```
npm run start
```

### RUN USING PM2

```
pm2 start npm --name=6529seize -- run start
```
