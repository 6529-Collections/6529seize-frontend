# Open Data Hub to Dataset Routes

Parent: [Open Data Index](README.md)

## Overview

This flow covers opening `/open-data` and moving into one dataset route.

## Entry Points

- Desktop sidebar: `Tools -> Open Data -> Open Data`
- Mobile sidebar: `Tools -> Open Data`
- Direct URL: `/open-data`

## Flow

1. Open `/open-data`.
2. Pick the dataset route card you need:
   - `Network Metrics`
   - `Meme Subscriptions` (can be hidden on native iOS outside US)
   - `Rememes`
   - `Team`
   - `Royalties`
3. Open the selected route and wait for the table to load.
4. Open a row link in a new tab, or move through pages when pagination is shown.

## Route Variants

- Direct links work for all dataset routes:
  - `/open-data/network-metrics`
  - `/open-data/meme-subscriptions`
  - `/open-data/rememes`
  - `/open-data/team`
  - `/open-data/royalties`
- `Meme Subscriptions` can be hidden on the hub for native iOS app users outside
  the US, but the direct route still works.
