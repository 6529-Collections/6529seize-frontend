Consolidations are distinct from delegations and are used to establish an on-chain message for Seize.io to treat two or more addresses as one for calculating wallet-related metrics such as total cards held, total unique cards held, and TDH.

This allows for a unified view of your NFTs in on-site galleries on seize.io and for any allowlist calculations.

You can set consolidations for either "All Collections" or just "The Memes" contract; if both are set, the "The Memes" contract will be used for The Memes.

Assuming three sample addresses:

  • Address A (vault) : 0xAcf42B85eCb77d9332584119FD78a3DE9953c2a0

  • Address B (transaction wallet) : 0x3558C942EeA9e9Bb9b1a6A02d272756EDD3A1Fe4

  • Address C (minting wallet) : 0xD0f03CB42Fe821c0654846E474c19484865c5B4a


Below is a process that assumes you have TAP and have granted Delegation Management (Sub-delegation) rights from your vault to your transaction wallet.
Before proceeding please make sure that you granted Delegation Management (Sub-delegation) rights from your vault to your transaction wallet.

Using Address B:

  1. Set a consolidation with Address A:


    • Navigate to the Delegation Center Section, click 'Register Consolidation,' choose 'The Memes' collection, enter wallet Address A, and execute the transaction.

  2. Set a consolidation with Address C:


    • Follow the same steps as above and enter wallet Address C.

  3. Set a consolidation on behalf of Address A with Address B and C (requires Delegation Management (Sub-delegation) from wallet A to wallet B):

    • Go to Delegation Center Section, click 'View/Manage' in 'The Memes Collection,' and select the Address of wallet A in Delegation Managers (Sub-Delegations) area.

    • Click 'Register Consolidation'.

  4. Register consolidation using Delegation Management (Sub-delegation) Rights:


    • Choose 'The Memes' collection and enter wallet Address B.

    • Execute the transaction.


    • Choose 'The Memes' collection and enter wallet Address C.

    • Execute the transaction.

Using Address C:

  1. Set a consolidation with Address A:


    • Navigate to the Delegation Center Section, click 'Register Consolidation,' choose 'The Memes' collection, enter wallet Address A, and execute the transaction.

  2. Set a consolidation with Address B:


    • Follow the same steps as above and enter wallet Address B.

  3. In the Delegation Center Section, click 'View/Manage' in 'The Memes Collection,' and check the consolidation status in the Consolidations area.

Notes
  • More than three addresses can have consolidations, but Seize.io will only count the last three delegations for consolidation purposes (to limit gaming).

  • Consolidations allow the safe usage of a 3-address architecture without any penalty in allowlists.

  • Within the context of The Memes, the purpose of consolidation is to allow you to consolidate any addresses that you own. We reserve the right to apply penalties to you or change the system if we believe you are gaming it (e.g. buying consolidation), such as removing your addresses from allowlists or altering the methodology, at our discretion, in any way.
