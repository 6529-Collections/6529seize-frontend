How to register an ENS domain name


  1. Navigate to https://ens.domains and click on the 'Go to app' button.

  2. In the next page, search for a name and check if its available. If it's available, click on the Available button to proceed.

  3. In the next page you can set the registration duration for the domain name and choose your payment method.
    When you select the Ethereum payment method it's recommended to enable the Use of the domain name as a primary name.
    If you select the Credit or debit card method, then you will need to revisit your profile and set the primary domain name.

  4. This is an optional step when you select the Ethereum as a payment method, you can Add more info to the profile or just Skip profile.

  5. This is the final step for registration. Please bear in mind that you will need to execute 2 transactions as per ens.domains instructions shown below:

    • Complete a transaction to begin the timer

    • Wait 60 seconds for the timer to complete

    • Complete a second transaction to secure your name by clicking the Finish button

  6. Once the 2nd transaction is executed successfully your name will be registered.


How to set a primary name


  1. Visit the app and click on 'My Names'

  2. Choose a domain name that you own

  3. Click on the 'Set as primary name' button

  4. Execute the transaction and your name will be set as a primary domain for your ETH address


How to create a subdomain


  1. Click on 'My Names' and select a name from the list.

  2. Click on the Subdomains Tab

  3. Then click on the New subname button

  4. Add a name, click Next and execute the transaction

  5. Click on the newly registered subdomain

  6. Now it depends on you how to use the newly created subdomain:

    • You can follow the instructions to set it as a primary name for your current ETH address

    • You can transfer it to another wallet and set it as a primary name from that ETH address.

    • You can create records and add an ETH address that will be resolved but its recommended to set it as a primary name so as applications can lookup the name when some searches it using the ETH address.
