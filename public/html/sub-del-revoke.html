How to revoke a delegation using sub-delegation rights?


Select a collection from the main page of the Delegation Center and navigate to the Sub-delegation area. If you have an incoming sub-delegation then certain actions will appear on the Sub-delegations area. To revoke a delegation using sub-delegation rights select the delegator address for which you want to revoke the delegation and then click the Revoke button. On the form select the collection for which you want to revoke a delegation on behalf of the Delegator, insert the wallet address and execute the transaction.
