How to update a delegation/sub-delegation/consolidation?


Select a Collection from the main page of the Delegations Center. On the Manage/View Collection page locate the address that you want to update, select it and click on the Update button next to it. On the form insert the new address, select an expiry date, specify if the new updated delegation refers to all tokens owned by the delegator or a specific token and execute the transaction.
